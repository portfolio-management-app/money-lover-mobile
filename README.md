### Naming convention:

- function and variable: Camel case

```ts
const handleInput = () => {};
let initNumber = 5;
```

- Type and Interface, Model, Component and Store: Title case:

```ts
interface AuthenticationProps {}

type Validate = {};

export const TextContainer = () => {};
```
